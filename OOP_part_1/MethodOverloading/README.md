\# Method Overloading



This folder contains Java programs demonstrating \*\*method overloading\*\*:



1\. \*\*Sum.java\*\* - sum() method with 2 or 3 parameters.

2\. \*\*Area.java\*\* - area() method for square, rectangle, and circle with different parameters.

3\. \*\*Max.java\*\* - max() method for 2 or 3 numbers.



\*\*Note:\*\* Method overloading allows multiple methods in the same class with the same name but different parameter lists.
