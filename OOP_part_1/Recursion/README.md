# Recursion in Java

This folder contains basic Java programs demonstrating recursion.

## Programs Included

1. FactorialRecursion.java
2. FibonacciRecursion.java
3. SumOfDigits.java
4. ReverseStringRecursion.java

## Key Concepts

- Recursion is a function calling itself
- A base condition is mandatory to avoid infinite calls
- Problems are broken into smaller sub-problems
- Useful for mathematical and string problems
